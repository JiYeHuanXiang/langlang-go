# LangLang-Go

> 项目来自 [redlang](https://github.com/super1207/redreply)，为方便想法实施而进行重制。

LangLang-Go 是一个跨平台聊天机器人自定义问答系统，使用 Go 语言重写。

原项目（RedLang/RedReply）使用 AGPL-3.0 许可，本重制版使用 **BSD 3-Clause** 许可。

感谢原作者的创意与开放态度。

## 全新的内容

- 更开放的开源协议！
- 新加入了测试模式
  - 本地模拟其他平台的输入
- 增加lua支持
- 加入JS支持
- 全新UI(还在修改，非最终方案)
- 手动发送功能
- 等待创意中
